# Camera, Cursor, And Editing Plan

## Camera Direction
- Use slow push-ins on chapter slides.
- Use zoom crops for live UI focus areas: command bar, map nodes, AI panel, incident queue, volunteer panel, mobile layout.
- Use pullbacks after each focused interaction to re-establish spatial context.
- Avoid shaky movement and constant panning.

## Cursor Plan
- Cursor starts off-screen, enters slowly, pauses before clicks, and exits after each interaction.
- Clicks include a green ripple and short highlight glow.
- Hover moments are held long enough for viewers to understand intent.
- Cursor path is curved and eased, not linear.

## Editing Plan
- 1080p MP4 output, 60 FPS.
- Use the existing PPTX as chapter material between live website sections.
- Use the live local website as the main visual proof.
- Use animated callouts only where they improve comprehension.
- Use focus blur/spotlight effects around dense UI areas.
- Use warm female narration with presentation pacing.
